# SECURITY.md

## Reporting

Report suspected vulnerabilities privately to the repository owner. Do not open a public issue containing exploit details, credentials, or private data.

## Automation policy

Pull-request content is untrusted. Workflows must declare least-privilege permissions, avoid automatic force-pushes, and require human review before applying AI-generated changes. Repository contents must not be sent to external AI services automatically. Rotate any exposed secret immediately and review workflow logs and recent commits.
